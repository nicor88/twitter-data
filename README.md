# twitter-data
Tools to collect and analyse twitter streams

## Dependencies
*   Docker
*   Node.js
*   npm

## Node.js tracker
The node.js tracker save the tweets based on the specified keywords into crateDB
To run the tracker move to nodejs-crateDb folder and run:
<pre>
docker-compose up
npm run tracker
</pre>
